# Config

Helpers to manage dict types configurations.

Version: 0.0.25

## Requirements

- Python >=3.6

## Dependencies

- pip-tools
- pydantic
- pytest
- wexample-helpers==0.0.25

## Installation

```bash
pip install wexample-config
```
