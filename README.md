# wexample-config

Helpers to manage dict types configurations.

Version: 0.0.50

## Requirements

- Python >=3.10

## Dependencies

- pydantic>=2,<3
- wexample-helpers==0.0.66

## Installation

```bash
pip install wexample-config
```
